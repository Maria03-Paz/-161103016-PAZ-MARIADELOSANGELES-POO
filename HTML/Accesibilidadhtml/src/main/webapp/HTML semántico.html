HTML semántico

HTML semántico significa usar elementos HTML correctos para su propósito correcto tanto como sea posible.

Si necesita un botón, use el elemento del botón y no un div:

Haga click en mi

Tiene mucho más sentido usar un elemento de botón: tiene un estilo más adecuado que el div, y es enfocable y se puede hacer clic de forma predeterminada.

Texto alternativo

El atributo alt debe reflejar el contenido de la imagen, para que los usuarios que no puedan ver la imagen comprendan qué contiene la imagen:

Si un navegador no puede encontrar la imagen, mostrará el texto alternativo:

Escribe buenos enlaces

Un enlace debe explicar claramente qué información obtendrá el lector haciendo clic en ese enlace.

Ejemplos de enlaces malos:

  • haga clic aquí
  • Lee mas..
  • Compra entradas para Marte aquí

Ejemplos de buenos enlaces:

  • Obtenga más información sobre el lenguaje HTML
  • Lea más sobrehow to eat healthy
  • cómo comer sano Compra entradas para Marte aquí

Tenga en cuenta que estos son solo ejemplos y no se puede hacer clic en ellos.
